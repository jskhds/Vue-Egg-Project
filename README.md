# Vue-Egg-Project
前后端分离的内容管理系统  	  
	1.技术栈：  
	  前  端：Vue + Element UI   
	  后  端：Egg + Nunjucks  + MySQL + Sequlize  
	2.技术要点：     
	  前端：  
	  ·使用 Element UI 实现页面快速布局  
	  ·使用Vue-router实现路由配置，重写vueRouter.prototype.push避免路由报错  
	  后端：  
	  ·使用Sequlize 操作 MySQL 数据库  
	  ·通过 egg 框架进行统一开发，提升效率
